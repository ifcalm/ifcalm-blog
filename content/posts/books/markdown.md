### 标题语法

要创建标题，请在词语前面添加`#`, `#`的数量代表了标题的级别：例如，添加三个`#`表示创建一个三级标题。   
使用规则：用一个空格在 `#` 和 `标题` 之间进行分隔。

案例：
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 段落语法

要创建段落，请使用`空白行`将一行或多行文本进行分隔。

案例：
```
寺庙里有个年轻小和尚，最近总是闷闷不乐，扫地时一脸苦相。
老方丈看他皱眉扫地，问：“徒儿，你为何如此苦恼？” 
小和尚叹气道：“师父，我想不通。人生到底有什么意义？每天扫地、吃斋、打坐，到底图个啥？” 
老方丈没直接回答，笑着说：“这样吧，我来问你三个问题。” 小和尚点头：“您请问。”

老方丈问：“你扫地，是为了让地干净，还是为了显得你很勤劳？” 
小和尚答：“当然是为了让地干净。” 
老方丈又问：“那地若明天又脏了，今日之扫，是否就没有意义？” 
小和尚犹豫一下：“也不能这么说，总得有人扫。”

老方丈点头：“所以，人生就像扫地，不是扫完就干净一辈子，
而是你每天扫，它每天净。若你心里净了，这世间也就清静了。” 
小和尚听完沉默良久，忽然眉头舒展，笑了。
```

### 换行语法

在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行。

案例：
```
人生就像扫地。   // 行末加3个空格
不是扫完就干净一辈子。   
而是你每天扫,它每天净。   
若你心里净了，这世间也就清静了。   
```

 ### 强调语法

#### 粗体

要加粗文本，请在词语的前后各添加两个星号（**）。

而是你**每天扫**，它每天净。若你**心里净**了，这世间也就清静了。

```
而是你**每天扫**，它每天净。若你**心里净**了，这世间也就清静了。
```

#### 斜体

要用斜体显示文本，请在词语前后添加一个星号（*）。

若你心里净了，这世间也就*清静*了。

```
若你心里净了，这世间也就*清静*了。
```

#### 粗体和斜体

要同时用粗体和斜体突出显示文本，请在词语的前后各添加三个星号（***）。

若你心里净了，这世间也就***清静***了。

```
若你心里净了，这世间也就***清静***了。
```

### 引用语法

要创建块引用，请在段落前添加一个 `>` 符号。

> 相濡以沫，不如相忘于江湖。
```
> 相濡以沫，不如相忘于江湖。
```

**多个段落的块引用：**   
块引用可以包含多个段落。为段落之间的空白行添加一个 `>` 符号。

> 朝菌不知晦朔，蟪蛄不知春秋。
> 
> 大知闲闲，小知间间；大言炎炎，小言詹詹。

```
> 朝菌不知晦朔，蟪蛄不知春秋。
> 
> 大知闲闲，小知间间；大言炎炎，小言詹詹。
```

**嵌套块引用：**   
块引用可以嵌套。在要嵌套的段落前添加一个 `>>` 符号。

> 大鹏一日同风起，扶摇直上九万里。
> 
>> 《庄子·逍遥游》

```
> 大鹏一日同风起，扶摇直上九万里。
> 
>> 《庄子·逍遥游》
```

**带有其它元素的块引用：**   
块引用可以包含其他 Markdown 格式的元素。

> **庄子·逍遥游**
> 
> - 独与天地精神往来，而不傲倪于万物。
> - 无用之用，方为大用。
> 
> 夫哀莫大于心死，而*人死亦次之*。

```
> **庄子·逍遥游**
> 
> - 独与天地精神往来，而不傲倪于万物。
> - 无用之用，方为大用。
> 
> 夫哀莫大于心死，而*人死亦次之*。
```

### 列表语法

#### 有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点（.）。

1. 天地有大美而不言。
2. 知其不可奈何而安之若命。
3. 人生天地之间，若白驹之过隙，忽然而已。
4. 鱼相忘于江湖，人相忘于道术。
5. 不谴是非，以与世俗处。

```
1. 天地有大美而不言。
2. 知其不可奈何而安之若命。
3. 人生天地之间，若白驹之过隙，忽然而已。
4. 鱼相忘于江湖，人相忘于道术。
5. 不谴是非，以与世俗处。
```

#### 无序列表

要创建无序列表，请在每个列表项前面添加中划线(-)。   
缩进一个或多个列表项可创建嵌套列表。

- 大鹏一日同风起，扶摇直上九万里。
- 汝安知鱼之乐？
- 不谴是非，以与世俗处。
    - 至人无己
    - 神人无功
    - 圣人无名
- 知其不可奈何而安之若命。

```
- 大鹏一日同风起，扶摇直上九万里。

- 不谴是非，以与世俗处。

    - 神人无功
    - 圣人无名
- 知其不可奈何而安之若命。
```

#### 在列表中嵌套其他元素

要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符。

- 大鹏一日同风起，扶摇直上九万里。
- 汝安知鱼之乐？
    > 《庄子·秋水》
- 知其不可奈何而安之若命。

### 代码语法

要将词语表示为代码，请将其包裹在反引号 (`) 中。

I like reading `Zhuangzi`.
```
I like reading `Zhuangzi`.
```

#### 转义反引号

如果要表示为代码词语中包含一个或多个反引号，则可以通过将词语包裹在双反引号(``)中。

Friends, ``I like reading `Zhuangzi`.``
```
Friends, ``I like reading `Zhuangzi`.``
```
