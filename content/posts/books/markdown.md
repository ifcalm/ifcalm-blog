### 标题语法

要创建标题，请在词语前面添加`#`, `#`的数量代表了标题的级别：例如，添加三个`#`表示创建一个三级标题。   
**使用规则：**用一个空格在 `#` 和 `标题` 之间进行分隔。

案例：
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 段落语法

要创建段落，请使用`空白行`将一行或多行文本进行分隔。

案例：
```
寺庙里有个年轻小和尚，最近总是闷闷不乐，扫地时一脸苦相。老方丈看他皱眉扫地，问：“徒儿，你为何如此苦恼？” 小和尚叹气道：“师父，我想不通。人生到底有什么意义？每天扫地、吃斋、打坐，到底图个啥？” 老方丈没直接回答，笑着说：“这样吧，我来问你三个问题。” 小和尚点头：“您请问。”

老方丈问：“你扫地，是为了让地干净，还是为了显得你很勤劳？” 小和尚答：“当然是为了让地干净。” 老方丈又问：“那地若明天又脏了，今日之扫，是否就没有意义？” 小和尚犹豫一下：“也不能这么说，总得有人扫。”

老方丈点头：“所以，人生就像扫地，不是扫完就干净一辈子，而是你每天扫，它每天净。若你心里净了，这世间也就清静了。” 小和尚听完沉默良久，忽然眉头舒展，笑了。
```

### 换行语法

在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行。

案例：
```
人生就像扫地。   // 行末加3个空格
不是扫完就干净一辈子。   
而是你每天扫,它每天净。   
若你心里净了，这世间也就清静了。   
```
