### 标题语法

要创建标题，请在词语前面添加`#`, `#`的数量代表了标题的级别：例如，添加三个`#`表示创建一个三级标题。   
使用规则：用一个空格在 `#` 和 `标题` 之间进行分隔。

案例：
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 段落语法

要创建段落，请使用`空白行`将一行或多行文本进行分隔。

案例：
```
寺庙里有个年轻小和尚，最近总是闷闷不乐，扫地时一脸苦相。
老方丈看他皱眉扫地，问：“徒儿，你为何如此苦恼？” 
小和尚叹气道：“师父，我想不通。人生到底有什么意义？每天扫地、吃斋、打坐，到底图个啥？” 
老方丈没直接回答，笑着说：“这样吧，我来问你三个问题。” 小和尚点头：“您请问。”

老方丈问：“你扫地，是为了让地干净，还是为了显得你很勤劳？” 
小和尚答：“当然是为了让地干净。” 
老方丈又问：“那地若明天又脏了，今日之扫，是否就没有意义？” 
小和尚犹豫一下：“也不能这么说，总得有人扫。”

老方丈点头：“所以，人生就像扫地，不是扫完就干净一辈子，
而是你每天扫，它每天净。若你心里净了，这世间也就清静了。” 
小和尚听完沉默良久，忽然眉头舒展，笑了。
```

### 换行语法

在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行。

案例：
```
人生就像扫地。   // 行末加3个空格
不是扫完就干净一辈子。   
而是你每天扫,它每天净。   
若你心里净了，这世间也就清静了。   
```

 ### 强调语法

#### 粗体

要加粗文本，请在词语的前后各添加两个星号（**）。

而是你**每天扫**，它每天净。若你**心里净**了，这世间也就清静了。

```
而是你**每天扫**，它每天净。若你**心里净**了，这世间也就清静了。
```

#### 斜体

要用斜体显示文本，请在词语前后添加一个星号（*）。

若你心里净了，这世间也就*清静*了。

```
若你心里净了，这世间也就*清静*了。
```

#### 粗体和斜体

要同时用粗体和斜体突出显示文本，请在词语的前后各添加三个星号（***）。

若你心里净了，这世间也就***清静***了。

```
若你心里净了，这世间也就***清静***了。
```

### 引用语法

要创建块引用，请在段落前添加一个 `>` 符号。

> 相濡以沫，不如相忘于江湖。
```
> 相濡以沫，不如相忘于江湖。
```

**多个段落的块引用：**   
块引用可以包含多个段落。为段落之间的空白行添加一个 `>` 符号。

> 朝菌不知晦朔，蟪蛄不知春秋。
> 
> 大知闲闲，小知间间；大言炎炎，小言詹詹。

```
> 朝菌不知晦朔，蟪蛄不知春秋。
> 
> 大知闲闲，小知间间；大言炎炎，小言詹詹。
```

**嵌套块引用：**   
块引用可以嵌套。在要嵌套的段落前添加一个 `>>` 符号。

> 大鹏一日同风起，扶摇直上九万里。
> 
>> 《庄子·逍遥游》

```
> 大鹏一日同风起，扶摇直上九万里。
> 
>> 《庄子·逍遥游》
```

**带有其它元素的块引用：**   
块引用可以包含其他 Markdown 格式的元素。

> **庄子·逍遥游**
> 
> - 独与天地精神往来，而不傲倪于万物。
> - 无用之用，方为大用。
> 
> 夫哀莫大于心死，而*人死亦次之*。

```
> **庄子·逍遥游**
> 
> - 独与天地精神往来，而不傲倪于万物。
> - 无用之用，方为大用。
> 
> 夫哀莫大于心死，而*人死亦次之*。
```

### 列表语法

#### 有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点（.）。

1. 天地有大美而不言。
2. 知其不可奈何而安之若命。
3. 人生天地之间，若白驹之过隙，忽然而已。
4. 鱼相忘于江湖，人相忘于道术。
5. 不谴是非，以与世俗处。


### 代码语法

要将词语表示为代码，请将其包裹在反引号 (`) 中。

I like reading `Zhuangzi`.
```
I like reading `Zhuangzi`.
```

#### 转义反引号

如果要表示为代码词语中包含一个或多个反引号，则可以通过将词语包裹在双反引号(``)中。

Friends, ``I like reading `Zhuangzi`.``
```
Friends, ``I like reading `Zhuangzi`.``
```

### 分隔线语法

要创建分隔线，请在单独一行上使用三个下划线 (---)，并且不能包含其他内容。  
为了兼容性，请在分隔线的前后均添加空白行。

相濡以沫，不如相忘于江湖。

---

人生天地之间，若白驹之过隙，忽然而已。

```
相濡以沫，不如相忘于江湖。

---

人生天地之间，若白驹之过隙，忽然而已。
```

### 链接语法

超链接语法代码：`[超链接显示名](超链接地址 "超链接title")`   
链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。   
链接title是当鼠标悬停在链接上时会出现的文字，这个title是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。

这是一个超链接：[诸子百家](https://books.ifcalm.org "诸子百家")
```
这是一个超链接：[诸子百家](https://books.ifcalm.org "诸子百家")
```

#### 网址和Email地址

使用尖括号可以很方便地把URL或者email地址变成可点击的链接。

诸子百家：<https://books.ifcalm.org/>   
我的邮箱：<ifcalm.ok@gmail.com>

```
诸子百家：<https://books.ifcalm.org/>   
我的邮箱：<ifcalm.ok@gmail.com>
```

#### 带格式化的链接

强调链接, 在链接语法前后增加星号。要将链接表示为代码，请在方括号中添加反引号。

我喜欢看 **[诸子百家](https://books.ifcalm.org)**   
他也喜欢看 *[诸子百家](https://books.ifcalm.org)*   
她也喜欢看 [`诸子百家`](https://books.ifcalm.org)


### 表格语法

要添加表，请使用三个或多个连字符（---）创建每列的标题，并使用管道（|）分隔每列。   
竖线（|）在英文状态下输入。

|       逍遥游           |     齐物论               |
| ---------------       | ----------------        |
|北冥有鱼, 其名为鲲        | 彼亦一是非, 此亦一是非      |
|小知不及大知, 小年不及大年 | 天地与我并生, 而万物与我为一 |


### 任务列表语法

任务列表可以创建带有复选框的项目列表。在支持任务列表的Markdown应用程序中，复选框将显示在内容旁边。要创建任务列表，请在任务列表项之前添加破折号(-)和方括号`[ ]`，并在`[ ]`前面加上空格。要选择一个复选框，请在方括号`[x]`之间添加 `x`。

- [x] 天地有大美而不言
- [ ] 乘物以游心，托不得已以养中
- [ ] 无用之用，方为大用

```
- [x] 天地有大美而不言
- [ ] 乘物以游心，托不得已以养中
- [ ] 无用之用，方为大用
```
